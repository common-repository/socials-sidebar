== Description ==

Social sidebar is a widget that fits into your sidebar and allows users to share your blog on various social networking sites.

== Installation ==

1. Upload `socialssidebar-media` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings => Socials Sidebar Settings and input your social media information
4. Place the Socials Sidebar widget in the desired widget area
5. Enjoy!

== Frequently Asked Questions ==

= How does this widget work? =

With this widget you simple choose which social buttons to display on your site, just enter your username for each of the sites and the widget will only show buttons for site you have an account on.
